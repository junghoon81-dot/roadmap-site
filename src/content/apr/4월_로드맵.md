# 김과외 4월 로드맵

> 4월 중순~말 실행 범위. 현욱님 기획 3개 파트.

3월 패치(4/13 배포)에 이어, 4월 남은 기간에는 현욱님이 준비한 3개 파트를 실행합니다. 매출 향상 포인트 8개, SEO 프로그래매틱 랜딩, 선생님 편의성 강화가 핵심입니다.

---

## 1. 매출 향상 포인트 8개

퍼널 곳곳에서 빠지는 매출을 잡는 8가지 시책입니다.

| # | 기능 | 내용 | 배포 목표 |
|---|------|------|----------|
| A | 성사 누락 잡기 | 채팅 후 미성사 학부모에게 알림톡 → 웹뷰 체크리스트. 30일·3회 캡 | 4월 |
| B | 시범→정규 전환 | 시범 후 14일+ 미전환 학부모 알림톡. 2회·2주 간격 | 4월 |
| C | FOMO 성사 알림 | 고민 중 선생님이 타인과 성사 시 즉시 푸시 | 4월 |
| D | 리뷰 도착 알림 | 채팅한 선생님에게 만점 후기 시 학부모 푸시 | 4월 |
| E | 공고 이탈 복구 | 작성 중 이탈 24시간 후 푸시 → 폼 복원 | 4월 |
| F | 선생님 부스트 | 앱 접속 시 인앱 팝업 → 화상과외 추가 + 추천 과목 추가 | 4월 |
| G | 휴면 깨우기 | 90일+ 미활동 학부모 → 맞춤 선생님 리스트 알림톡 | 4월 (A/B 선행) |
| H | 수업 결과 설문 | 성사 3개월 후 알림톡 → 분기형 설문 + 스벅 기프티콘 | 4월 말 |

H(수업 결과 설문)는 "성사 3개월 후" 1회 시점에 집중합니다. 5월 Phase 1에서 이를 확장하여 2주/1개월/2개월/종료 4시점 전 구간 설문 체계로 고도화할 예정입니다.

## 2. SEO 프로그래매틱 랜딩

| 항목 | 내용 | 배포 목표 |
|------|------|----------|
| 선생님 프로필 비로그인 공개 | 소개서·후기·경력 전부 구글 인덱싱 (사진/영상만 로그인 트리거) | 4월 말 |
| 4종 SEO 페이지 | 선생님 찾기(학부모용) / 과외비로 찾기(학부모용) / 선생님 가입유인(선생님용) / 프로필 SEO 오픈 | 4월 말 |
| 10만+ 페이지 자동 생성 | DB 데이터 + 과목/지역 변수 조합, zero human-in-the-loop | 4월 말 |

3월 패치에서 도입한 소개서 구조화 + AI 자연어 검색과 연결됩니다. 소개서가 채워질수록 SEO 페이지의 콘텐츠 품질이 올라갑니다.

## 5. 5월 연결

4월 배포 항목들이 5월 로드맵의 Phase 1으로 자연스럽게 이어집니다. 특히 H(수업 결과 설문)가 5월 ① 매주 체크인 + 고정 시점 설문의 기초가 되고, SEO 랜딩이 5월 ② 추천이유카드 v2-A의 반영 대상이 됩니다.
