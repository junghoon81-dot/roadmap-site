# 김과외 3월 로드맵

> **축③ 성사율 올리기**의 첫 발판. 3월 개발 → **4/13 배포 예정.**

지금 김과외의 성사율이 막히는 이유는 단순합니다. 학부모가 "이 선생님이 나한테 맞는지" 판단할 재료가 부족합니다. 프로필은 대부분 비어있고, 검색은 필터 조합뿐입니다.

3월 패치는 이 문제의 기반을 까는 작업입니다. 소개서를 구조화해서 선생님 데이터를 채우고, AI 자연어 검색으로 그 데이터를 매칭에 쓸 수 있게 만듭니다. 동시에 공고 UX를 개편해서 학부모가 더 나은 의사결정을 할 수 있도록 정보를 강화합니다. 수업료 납부 요청서 자동발송과 현금영수증 자동발급은 **축① 관계 쌓기**의 첫 접점이기도 합니다.

---

## 1. 공고 개편 — 경쟁강도 + 정보 강화

학부모가 공고를 올린 뒤 받는 제안이 31건을 넘으면 답장률과 성사율이 급격히 하락합니다. 선생님 입장에서도 치열한 공고에 제안을 보내는 건 시간 낭비입니다. 양쪽 모두에게 경쟁 상황을 투명하게 보여주기 위해 경쟁강도 배너를 도입합니다.

| 경쟁강도 | 조건 |
|----------|------|
| 여유 | 받은 제안 10건 이하 |
| 적정 | 받은 제안 11–30건 |
| 치열 | 받은 제안 31건 이상 |

과외찾기 목록에서도 경쟁강도를 바로 확인할 수 있습니다. 치열한 공고에도 제안은 가능하지만, 여유·적정 위주로 제안하는 것을 권장합니다.

### 공고 정보 강화

선생님이 제안을 보낼 때 판단할 수 있는 정보를 늘립니다. 지금까지는 과목과 지역 정도만 보였지만, 이제 학생의 현재 상황, 목표, 선호하는 선생님 조건, 과목별 보완 파트까지 공고에 표시됩니다. 이 정보가 풍부해질수록 선생님이 "내가 가르칠 수 있는 학생"을 골라서 제안할 수 있고, 성사율이 올라갑니다.

추가로 성사 후 2주 뒤부터 후기 작성 요청 알림톡을 발송하고, 다른 과목을 빠르게 찾을 수 있는 원클릭 공고 수정 기능을 도입합니다. 대면과외만 설정한 학부모 중 제안이 적은 분께는 화상과외 추가를 안내합니다.

## 2. 소개서 구조화

### 소개서 신규 항목

| 항목 | 내용 |
|------|------|
| 수업 요일/시간대 | 기존 요일만 → 시간대까지 설정 가능. 최근 업데이트 시점도 소개서에 표시 |
| 수업 목적 | 내신 대비, 수능/모고 대비 등 직관적으로 검색 가능하도록 |
| 특히 자신있는 파트 | 과목별 세부 파트 선택 (예: 생명과학 → 유전, 근수축, 막전위·신경전도) |
| 추가 강점 정보 | AI 자연어 검색 대응용 자유 입력 필드 |
| 미완성 항목 모아보기 | 내 소개서 관리에 미완성 필터 추가 |

소개서 완성도 80% 이상 선생님의 성사율은 80% 미만 선생님 대비 약 **12배** 높습니다. 소개서 구조화는 바로 아래 AI 자연어 검색의 데이터 기반이 됩니다.

## 3. AI 자연어 검색

이번 패치에서 가장 중요한 신규 기능입니다. 기존 필터 검색의 한계를 넘어, 학부모가 원하는 조건을 문장으로 입력하면 AI가 최적의 선생님을 추천합니다.

### 왜 중요한가

김과외에는 65만 명의 선생님이 등록되어 있지만, 학부모가 원하는 조건(지역 + 과목 + 수업 스타일 + 경험)을 필터 조합으로 표현하기 어렵습니다. "서초구 고등 여자 수학 선생님, 수포자도 잘 가르치는 분"처럼 자연어로 검색하면, 필터로는 잡을 수 없는 정성적 조건까지 매칭할 수 있습니다.

### 작동 방식

학부모가 조건을 문장으로 입력하면, AI가 다음 6가지 소스를 종합 분석하여 적합도 높은 순서로 추천합니다:

| 분석 소스 | 내용 |
|-----------|------|
| 수업 과목 | 등록 과목 + 세부 파트(§2 신규 항목) |
| 전문 분야 | 소개서 자유 입력 + 자신있는 파트 |
| 지도 경력 | 경력 연수 + 수업 이력 |
| 후기 | 125K건 리뷰에서 키워드 매칭 |
| 성사 건수 | 매칭 실적 기반 신뢰도 |
| 소개서 전문 | 구조화된 항목 + 자유 텍스트 |

### 기존 검색과의 관계

- 기존 선생님 목록 + 필터는 그대로 유지
- AI 자연어 검색은 별도 메뉴로 운영
- 적합도가 동일하면 기존 선생님 랭킹 순으로 정렬
- 4월 SEO 프로그래매틱 랜딩과 연결: 소개서가 채워질수록 검색 품질 상승
- 5월 추천이유카드 v2-A의 데이터 기반이 됨

## 4. 수업료 납부 요청서 자동발송

**축① 관계 쌓기**의 시작점입니다. 지금 김과외는 매칭 성사 후 관계가 끊깁니다. 선생님과 학부모가 카톡으로 빠져나가고, 수업료도 계좌이체로 주고받습니다. 수업료 납부 요청서 자동발송은 이 끊어진 관계에 김과외가 다시 끼어드는 첫 번째 접점입니다.

수업료, 계좌번호, 발송일자를 입력하면 김과외 카톡채널로 납부 요청서가 자동 발송됩니다. 계좌 정보는 발송 후 30일 자동 만료. 김과외 비회원(외부 과외 학생/학부모)에게도 발송 가능하고, 여러 학생에게 한 번에, 매월 N일 정기 발송까지 됩니다.

이 알림톡 발송 + 자동 로그인 인프라는 4월 매출 장치(성사 누락 알림톡, 시범전환 알림톡 등)와 5월 체크인 파이프라인에서 그대로 재활용됩니다.

## 6. 운영 — 학원법 개정 + 과목 신설

### 학원법 개정 대응

학원법 제12조의2 신설(유아 대상 모집시험·평가 금지)에 따라 관련 4개 과목(영유입시, 영유레테, 학원입시, 영재원입시)의 제공을 중단합니다. 기존 성사된 수업·채팅방·후기 등 데이터는 그대로 유지됩니다.

### 과목 신설

고객지원팀으로 접수된 1–3월 신규 과목 신청 내역을 검토하여 **총 12개 과목**이 새로 신설되었습니다.

### 분쟁해결 솔루션

1–3월 동안 총 **61건**의 분쟁해결 솔루션을 제공했으며, 선생님 약 55명·학생/학부모 약 95명에 대해 이용제한 조치를 완료했습니다.
