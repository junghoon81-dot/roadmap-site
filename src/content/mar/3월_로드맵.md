# 김과외 3월 로드맵

> 3월 개발 → 4/13 배포 완료.

공고 UX 개편, 소개서 구조화, AI 자연어 검색 도입, 수업료 납부 자동화, 현금영수증 자동발급, 학원법 개정 대응까지 한 사이클로 묶어 실행한 패치입니다.

---

## 1. 공고 개편 — 경쟁강도 + 정보 강화

### 경쟁강도 배너 도입

한 공고에 제안이 31건을 넘으면 답장률과 성사율이 급격히 하락합니다. 이를 보완하기 위해 각 공고에 경쟁강도 배너를 도입했습니다.

| 경쟁강도 | 조건 |
|----------|------|
| 여유 | 받은 제안 10건 이하 |
| 적정 | 받은 제안 11~30건 |
| 치열 | 받은 제안 31건 이상 |

과외찾기 목록에서도 경쟁강도를 바로 확인할 수 있습니다. 치열한 공고에도 제안은 가능하지만, 여유·적정 위주로 제안하는 것을 권장합니다.

### 공고 정보 강화

- 학생의 학년, 계열 정보 등 선생님에게 중요한 정보를 강조
- 학생의 현재 상황 및 목표, 선호하는 선생님 조건, 과목별 집중 보완 파트 등 새로 추가
- 과외 성사 후 2주 후부터 후기 작성 요청 알림톡 발송
- 성사 후 다른 과목도 빠르게 찾을 수 있도록 **원클릭 공고 수정** 기능 도입
- 대면과외만 설정한 학부모 중 제안이 적은 분께 화상과외 추가 안내 알림톡 발송

## 2. 소개서 구조화 + AI 자연어 검색

### 소개서 신규 항목

| 항목 | 내용 |
|------|------|
| 수업 요일/시간대 | 기존 요일만 → 시간대까지 설정 가능. 최근 업데이트 시점도 소개서에 표시 |
| 수업 목적 | 내신 대비, 수능/모고 대비 등 직관적으로 검색 가능하도록 |
| 특히 자신있는 파트 | 과목별 세부 파트 선택 (예: 생명과학 → 유전, 근수축, 막전위·신경전도) |
| 추가 강점 정보 | AI 자연어 검색 대응용 자유 입력 필드 |
| 미완성 항목 모아보기 | 내 소개서 관리에 미완성 필터 추가 |

소개서 완성도 80% 이상 선생님의 성사율은 80% 미만 선생님 대비 약 **12배** 높습니다.

### AI 자연어 검색 도입

"서초구 고등 여자 수학 선생님, 수포자도 잘 가르치는 분"처럼 원하는 조건을 문장으로 검색하면, AI가 수업 과목·전문 분야·지도 경력·후기·성사 건수·소개서 내용을 종합 분석하여 적합도 높은 순서로 추천합니다.

- 적합도가 동일하면 기존 선생님 랭킹 순으로 정렬
- 기존 선생님 목록 + 필터는 그대로 유지, AI 검색은 별도 메뉴로 운영

## 3. 수업료 납부 요청서 자동발송

매번 수업료를 직접 요청해야 하는 부담을 덜기 위해 자동 발송 기능을 도입했습니다.

- 수업료, 계좌번호, 발송일자 입력 → 정해진 일자에 김과외 카톡채널로 납부 요청서 발송
- 계좌 정보는 발송 후 30일 자동 만료
- 김과외 회원 + 비회원(외부 과외 학생/학부모)에게도 발송 가능
- 여러 학생에게 한 번에 발송 가능, 매월 N일 정기 발송 설정 가능
- 멤버십 레벨별 월간 발송 한도 (Lv.0: 3건 ~ Lv.6: 100건), 매월 1일 초기화

## 5. 운영 — 학원법 개정 + 과목 신설

### 학원법 개정 대응

학원법 제12조의2 신설(유아 대상 모집시험·평가 금지)에 따라 관련 4개 과목(영유입시, 영유레테, 학원입시, 영재원입시)의 제공을 중단합니다. 기존 성사된 수업·채팅방·후기 등 데이터는 그대로 유지됩니다.

### 과목 신설

고객지원팀으로 접수된 1~3월 신규 과목 신청 내역을 검토하여 **총 12개 과목**이 새로 신설되었습니다.

### 분쟁해결 솔루션

1~3월 동안 총 **61건**의 분쟁해결 솔루션을 제공했으며, 선생님 약 55명·학생/학부모 약 95명에 대해 이용제한 조치를 완료했습니다.
